# Thermal Conductivity Simulation

## Overview

This project demonstrates thermal conductivity and heat transfer through different metal materials using a real-time 3D simulation.

## Features

* Real-time heat transfer simulation
* Interactive material selection
* Dynamic temperature visualization
* Heat particle effects
* Start and pause controls
* Live temperature monitoring panel

## Technologies Used

* Babylon.js
* TypeScript
* Vite
* Babylon GUI

## Materials Simulated

* Copper
* Iron
* Steel

## How It Works

The simulation models heat diffusion through a metal rod. Different materials use different conductivity values, affecting how quickly heat travels from the heater to the cooler.

## User Interaction

Users can:

* Start or pause the simulation
* Change materials
* Observe temperature changes visually
* Monitor live sensor data

## Outcome

The project provides an interactive engineering visualization for understanding thermal conductivity concepts in an engaging 3D environment.
